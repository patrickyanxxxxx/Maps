# iOS 27 Maps 历史版本

此目录保存开发过程中产生的旧模块和旧脚本，仅用于对比、诊断和回归测试。

- `legacy/`：旧 Egern 模块、早期多客户端模块、独立 CN/国际模式、Worker Rewrite 模块和诊断模块。
- `assets/`：旧 request/response bundle、历史卫星路由脚本和诊断脚本。

日常使用请返回上一级 `modules/`，选择当前稳定版：

- `iRingo.Maps.yaml`：Egern
- `iRingo.Maps.sgmodule`：Surge
- `iRingo.Maps.plugin`：Loon
- `iRingo.Maps.srmodule`：Shadowrocket
- `iRingo.Maps.stoverride`：Stash
- `iRingo.Maps.snippet`：Quantumult X

不要同时启用历史版本和当前稳定版本。
